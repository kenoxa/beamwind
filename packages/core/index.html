Project @beamwind/core - v1.0.0

@beamwind/core

compiles Tailwind CSS like shorthand syntax into CSS at runtime

Read the docs | API | Change Log

The core theme is bare bones as most projects have their own colors, sizes, ... and naming system. You probably want use one of the following:

  • beamwind - using the tailwind default theme and preflight
  • @beamwind/system - using a semantic design system and preflight
  • @beamwind/play - combining the tailwind default theme, the semantic design system and preflight with auto-conversion of unknown theme values
  • Installation
  • Contribute
  • License

Installation

 
npm install @beamwind/core

And then import it:

import { bw } from '@beamwind/core'

Please refer to the main documentation for further information.
