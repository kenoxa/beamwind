Installation

 
npm install @beamwind/tailwind

Usage

 
import { tw } from '@beamwind/tailwind'

tw`text-purple-500`

For further customization setup is exported:

import { tw, setup } from '@beamwind/tailwind'

setup({
  colors: {
    'red-500': 'hotpink',
  },
})

tw`text-red-500` // will result in a hotpink background-color
